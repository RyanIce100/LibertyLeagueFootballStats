# LibertyLeagueFootballStats
This repo serves as documentation of how I created a Liberty League D3 Football dataset that encompasses the years 2013-2025. The data collected from 2013-2018 was from Logan Hansen (https://github.com/HansenRatings/DIII-Stats). All data and code will be available here for any who want to follow along. 

It is likely that there are many errors in this dataset due to the AI scraping code and the fact that I combed through it manually. If any are thinking about doing this process for other D3 Teams, I would not recommend doing it how I did it (mostly manually). If you catch any major errors with the dataset, please contact me/let me know.
